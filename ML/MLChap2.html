Exploratory Data Analysis (EDA)

CCS 247 - Machine Learning

Introduction to EDA

EDA is the initial process of analyzing and summarizing datasets to discover patterns, anomalies, and underlying structures, often using visual and statistical methods. It's a detective-like approach to understand the "story" hidden within the data.

Exploratory Data Analysis (EDA) is a philosophy for data analysis that employs a variety of techniques (mostly graphical) to maximize insight into a data set.

The key goals of EDA include:

  • Uncover underlying structure
  • Extract important variables
  • Detect outliers and anomalies
  • Test underlying assumptions

EDA in Machine Learning

Exploratory Data Analysis plays a crucial role in the machine learning workflow, helping to understand data before modeling.

Applications in ML

EDA helps ensure proper data quality, guides feature engineering, aids in model selection, detects bias, and helps prevent overfitting.

  • Data Quality - Ensures data is clean, consistent, and suitable for modeling.
  • Feature Engineering - Identifies relevant features and guides the creation of new ones.
  • Model Selection - Helps choose appropriate algorithms based on data characteristics.
  • Bias Detection - Uncovers potential biases that could affect model fairness.
  • Overfitting Prevention - Helps understand data complexity and avoid overfitting.

Key Stages of EDA

Exploratory Data Analysis follows a systematic approach consisting of several key stages.

The EDA Process

  • Data Collection - Gathering raw data from various sources.
  • Data Cleaning - Addressing missing values, duplicates, and errors.
  • Univariate Analysis - Examining individual variables.
  • Bivariate Analysis - Investigating relationships between two variables.
  • Multivariate Analysis - Analyzing relationships among multiple variables.
  • Visualization - Creating graphical representations of data.
  • Statistical Analysis - Applying statistical methods to quantify patterns and relationships.

Data Cleaning and Preprocessing

Data cleaning and preprocessing are essential steps to ensure that the data is ready for analysis and modeling.

Data Quality Issues

  • Missing Data - Data points that are absent or unknown
  • Dupicates - Identical or near-identical data entries
  • Outliers - Data points that deviate significantly from the rest of the data

Cleaning Data

  • Missing Data - handled through deletion, imputation, or use of "missing" indicators
  • Duplicates - can be handled by removing duplicate rows to avoid bias
  • Outliers - Can be detected by using box plots, scatter plots, z-scores, IQR (Interquartile Range). Can be handled through removal, transformation, or separate treatments.

Preprocessing Data

  • Data Type Conversion - Ensuring data types (integer, float, string, datetime) are correct
  • Standardization (Z-score scaling) - Transforms data to have zero mean and unit variance
    • \[z = \frac{x - \mu}{\sigma}\]
  • Normalization (Min-Max scaling) - Scales data to a specific range (e.g., 0 to 1)
    • \[y = \frac{x - \min(x)}{\max(x) - \min(x)}\]

Univariate Analysis

Univariate analysis involves examining individual variables one at a time. The two types of varibles are Categorical and Numerical variables.

Categorical Variables

Represent categories or labels, can be either Nominal or Ordinal

Mode - the most frequent category

    Visualization

    • Frequency Tables - Counting the occurrences of each category.
    • Bar Charts - Visualizing category frequencies.
    • Pie Charts - Showing the proportion of each category.

Numerical Variables

Represent quantities or measurements, can be either Continuous or Discrete

    Measurements of Central Tendency

    • Mean - average value
    • Median - middle value
    • Mode - most frequent value

    Measurements of Dispersion

    • Range - difference between maximum and minimum values
    • Variance - average squared deviation from the mean
    • Standard Deviation - square root of the variance
    • IQR (Interquartile Range) - range of the middle 50% of the data

    Visualization

    • Histograms - Showing the distribution of values
    • Box Plots - Displaying quartiles and outliers.
    • Density Plots - Smoothed representation of the distribution

Multivariate Analysis

Multivariate analysis examines relationships between multiple variables simultaneously.

Probability Distributions

  • Normal Distribution - Bell-shaped, symmetrical distribution.
  • Binomial Distribution - Probability of successes in a fixed number of trials.
  • Poisson Distribution - Probability of events occurring in a fixed interval.

Visualization

  • Pair Plots - Visualizing relationships between all pairs of numerical variables.
  • Heatmaps (Correlation) - Displaying correlation matrices for multiple variables.
  • Parallel Coordinates Plots - Visualizing relationships between multiple numerical variables.
  • 3D Scatter Plots - Visualizing relationships between three numerical variables.
